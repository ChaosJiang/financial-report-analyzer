# Chris Skills

Skills shared by Chris for improving daily work efficiency. Works with Claude Code / Amp.

## Installation

```bash
git clone https://github.com/ChaosJiang/chris-skills.git ~/.claude/skills/chris-skills
```

## Skills

| Skill | Description |
|-------|-------------|
| [chris-stock-master](skills/chris-stock-master/) | 深度分析全球上市公司财报（美/日/港/A股），提供估值分析及投资建议 |

## Usage

After installation, use skills directly in Claude Code:

```bash
# Analyze Apple stock
分析 AAPL

# Analyze Tencent
分析 0700.HK

# Analyze Kweichow Moutai
分析 600519.SH
```
